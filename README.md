# Telegram Bot

## Как использовать

1. Скопируй `.env.example` в `.env` и вставь свой токен бота.
2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запусти бота:
   ```bash
   python bot.py
   ```

Для деплоя на Render.com добавь переменную окружения `TELEGRAM_TOKEN` в настройках сервиса.
